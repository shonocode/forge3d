FORGE 3D Manual
GETTING STARTED
Quick Start

1. 左パネルからプリミティブを追加

2. ツールピル（ヘッダー）でツールを切り替えて編集

3. GLB Export / Save to Library で保存

4. ファイルをビューポートにドラッグ&ドロップでインポート

Camera & Viewport

Camera Controls:

左ドラッグ: 回転 / 右ドラッグ・2本指: パン / ホイール・ピンチ: ズーム

Camera Presets（ビューポート右上）:

F=Front, Bk=Back, R=Right, L=Left, T=Top, Bt=Bottom。クリックでビュー切替。

⊞ ボタンで正射影（Orthographic）/透視投影（Perspective）を切替。

Viewport Shading（左下ボタン）:

○ Solid: フラットライティング / ◇ Wire: ワイヤーフレーム / ◔ Matcap: マットキャップ / ● Textured: PBRマテリアル表示（デフォルト）

Mobile Controls

下部バー: +Prim, Undo, Dup, Del, Export, Save, Load

🔒 Cam: カメラロック — カメラ操作を無効化してギズモ操作に集中。もう一度タップで解除。

⊚ Multi: マルチセレクトモード — タップで複数メッシュを選択（Ctrl+クリック代替）。

Sculpt/Weight モード: ⇅ Invert / 〜 Smooth トグルボタンで Ctrl+ドラッグ / Shift+ドラッグ を代替。

パネルはスワイプで閉じられます（左パネル=左スワイプ、右パネル=右スワイプ）。

ギズモは自動的に1.5倍サイズで表示され、タッチ操作しやすくなっています。

MODELING
Primitives + CSG

左パネルからBox, Sphere, Cylinder, Torus, Plane, Cone, Tube, Disc, Icosphere, Capsule を追加。

2つのメッシュを選択（Ctrl+クリック）して CSG 演算:

∪ Union（結合）/ − Subtract（差分）/ ∩ Intersect（交差）

Select / Transform

クリックでメッシュ選択。Ctrl+クリック or 長押し(モバイル)で複数選択。

MOV/ROT/SCL でギズモが表示され、ドラッグで変形。右パネル Transform タブで数値入力も可能。

メッシュの寸法（幅×高さ×奥行き）は Transform タブの Dimensions に表示されます。

Mesh Tools

左パネル「Mesh Tools」セクション。メッシュを選択して使用。全てUndo対応。

Recalc Normals: 法線を再計算。Sculpt/CSG後のシェーディング修正に。

Flip Normals: 法線を反転。メッシュの裏表が逆の場合に。

Shade Smooth / Flat: なめらか / フラットシェーディング切替。

Auto Smooth ∠: 角度°より急な折り目だけハードエッジに（Blender の Auto Smooth 相当）。

Weld Vertices: 近接頂点を結合。CSGやインポート後の重複頂点除去に。

Center Origin: ピボットをジオメトリ中心に移動。リギング前の準備に。

Modifiers

右パネル Transform タブの「Modifiers」セクション。非破壊のジオメトリ変換。

Subdivision: ポリゴンを分割して滑らかに。Level 1-2。

Mirror: X/Y/Z軸でジオメトリをミラー。Merge ON で対称面の頂点を結合。

Array: ジオメトリをN回繰り返し配置。X/Y/Zオフセット指定。フェンスや階段に。

各モディファイアは ON/OFF トグル、Apply（確定）、Delete（削除）が可能。スタック順に評価されます。

Snap Settings

右パネル Transform タブの「Snap」セクション。ギズモ操作をグリッドに吸着。 Edit Mode のコンポーネントギズモ（頂点 / 辺 / 面の移動・回転・拡縮）にも適用される。

Position: 移動をスナップ（デフォルト: 0.5m刻み）

Rotation: 回転をスナップ（デフォルト: 15°刻み）

Scale: スケールをスナップ（デフォルト: 0.25刻み）

チェックボックスで有効化し、数値で刻み幅を調整。

Measurement

左パネル「Measure」セクション。

「Measure Distance」→ メッシュ上の2点をクリック → 距離（メートル）を表示。

オレンジのマーカーとラインで視覚表示。複数の測定を同時に表示可能。

「Clear All」で全測定を削除。

SURFACE
Material / PBR

右パネル Material タブ。メッシュを選択して物理ベースマテリアルを編集。

Albedo: ベースカラー。パレットまたはカラーピッカーで選択。

PBR: Metallic（金属感 0-1）/ Roughness（粗さ 0-1）/ Alpha（透明度 0-1）

Emissive: 発光色 + 強度。自己発光やグロー効果に。

Clear Coat: 光沢クリアレイヤー。車の塗装やコーティング表現に。

Sheen: 布のような光沢。ファブリックやベルベット表現に。

Transmission: 透過+屈折。IOR設定: Glass=1.5, Water=1.33, Diamond=2.42

Textures: Albedo / Normal / Metal-Rough / AO / Emissive の5スロット。画像をインポート。

Display: Wireframe トグル / Unlit トグル（ライティング無視）

Sculpt

メッシュを選択してSCPモードに切替。ドラッグで頂点を変形。

Ctrl+ドラッグ: 反転（Push↔Pull）/ Shift+ドラッグ: Smooth

右パネルでブラシの Size / Strength / Falloff を調整。

6ブラシ: Push（押す）, Pull（引く）, Smooth（平滑化）, Flatten（平坦化）, Pinch（つまむ）, Inflate（膨張）

モバイル: ⇅ Invert / 〜 Smooth トグルボタンで Ctrl/Shift を代替。

Paint

メッシュを選択してPNTモードに切替。メッシュ表面に直接ペイント。

Color / Size / Opacity / Eraser を右パネルで設定。

Clear Paint でテクスチャをリセット。GLBエクスポート時にテクスチャとして埋め込まれます。

Morph Targets

メッシュ選択 → Morph タブ → 「ターゲット有効化」で MorphTarget 追加。

メッシュを変形（Sculpt等）→「形状キャプチャ」で現在の形状を記録。

スライダーで補間。顔の表情変化やブレンドシェイプに使用。

RIGGING & ANIMATION
Bone / Skeleton

BONEモードでメッシュ表面をクリック → ボーンを配置。

既存ボーンをクリック → 次のクリックで子ボーンを追加（チェーン作成）。

「New Skeleton」でスケルトン作成。「Assign to Mesh」で選択メッシュにアタッチ。

ボーンのギズモを動かしてポーズ確認。ボーン名は右パネルで変更可能。

IK Constraints

右パネル Bone タブの「IK Constraint」セクション。

ボーンを選択 → 「Enable IK」チェックで逆運動学（IK）を有効化。

Chain Length: IKチェーンに含める親ボーンの数（1-5）。

末端ボーンを動かすとチェーン全体が自然に追従。手足のポーズに便利。

Weight Paint

スケルトンをアタッチしたメッシュを選択 → WGTモードに切替。

「Init Weight Data」でウェイトデータ初期化（必須）。

Bone タブでボーンを選択 → メッシュ上をドラッグでウェイト塗り。

モード: Add（加算）/ Subtract（減算）/ Smooth（平滑化）

Ctrl+ドラッグ: Subtract / Shift+ドラッグ: Smooth。モバイル: トグルボタン使用。

「Show/Hide Overlay」でヒートマップ表示（赤=1.0, 緑=0.5, 青=0.0）。

4ボーン/頂点まで。ウェイトは自動正規化（合計=1.0）。

Animation

ANMモードに切替 → 「New Clip」でアニメーションクリップ作成。

Timeline スライダーでフレーム選択 → ボーンをポーズ → 「Record Keyframe」。

「Record All Bones」で全ボーンの現在ポーズを一括記録。

「Play」でプレビュー再生。「Export JSON」でアニメーションデータ出力。

FPS / Max Frames / Loop Mode（Cycle/Constant）を設定可能。

GLBインポート時のアニメーションは Imported Animations セクションで再生可能。

Easing Curves

右パネル Anim タブの「Keyframe Edit」セクション。

キーフレーム間の補間曲線をイージングで制御。ドロップダウンから選択。

種類: Linear, Quad, Cubic, Elastic, Bounce（各 In/Out/InOut）

「Copy KF」でキーフレームをコピー → 別フレームで「Paste KF」で貼り付け。

SCENE & PROJECT
Layers

左パネル「Layers」セクション。メッシュをレイヤーで整理。

「+ New Layer」でレイヤー作成。クリックでアクティブレイヤーを切替。

新しいプリミティブやインポートはアクティブレイヤーに追加されます。

目のアイコンでレイヤーの表示/非表示を切替。シーン整理に便利。

Hierarchy / Parenting

左パネル「Hierarchy」セクション。

Set Parent: 2つ以上のメッシュを選択（Ctrl+クリック）→ 最後に選択したメッシュが子になる。

Clear Parent: 選択メッシュの親子関係を解除。

子メッシュは親のトランスフォーム（移動/回転/スケール）に追従します。

Lighting

右パネル Scene タブの「Lights」セクション。最大8ライト。

+ Point: 全方向に光を放つポイントライト。

+ Spot: 指定方向に円錐状の光を放つスポットライト。角度調整可能。

各ライト: Color（色）/ Intensity（強度 0-5）/ Range（範囲 1-50）を設定。

ライトの球体/コーンをクリックして位置を移動可能。

Environment / HDRI

右パネル Scene タブの「Environment」セクション。

Preset: Studio / Country からプリセットを選択。

Import HDRI: カスタム .hdr ファイルを読み込み。画像ベースライティング。

Intensity: 環境光の強度（0-3）。反射や間接照明に影響。

Skybox: 環境をビューポート背景として表示。

Shadows

右パネル Scene タブの「Shadows」セクション。

チェックボックスで影を有効化/無効化。

Quality: Low(512) / Medium(1024) / High(2048) — シャドウマップ解像度。

全メッシュが自動的に影を投影・受影します。

Save / Export / Import

GLB Export: シーン全体をGLBとしてダウンロード。スケルトン・ウェイト・テクスチャ・アニメーション含む。

OBJ Export: .obj + .mtl ファイルとしてダウンロード。マテリアル付きジオメトリのみ。

Save to Library: ブラウザ内ストレージ（OPFS/IndexedDB）に保存。Map Editorで再利用可能。

Load Model: ファイル選択ダイアログから GLB/glTF/OBJ を読み込み。

Drag & Drop: ファイルをビューポートに直接ドロップしてインポート。

Undo/Redo: ヘッダーの ↶/↷ ボタンまたは Ctrl+Z / Ctrl+Shift+Z。

Map Editor

Map タブ → 「Refresh Library」でライブラリ読み込み。

「+」ボタンでモデルをシーンに配置。ギズモで位置・回転・スケール調整。

「Export Layout JSON」でシーン配置データを出力。「Import Layout」で復元。

「Clear Scene」でシーン内の全インスタンスをクリア。

PWA

スマホのホーム画面に追加するとオフラインで使用可能。

データはブラウザ内に保持。デバイス間の転送は GLB Export → Load で。

Workflow Example

セットアップ → モデリング → リギング → アニメーション → エクスポート

1. Environment / Lighting でシーンを設定

2. プリミティブ追加 + CSG + Sculpt で形状作成

3. Material で PBR マテリアルを設定

4. Paint でテクスチャを描画

5. Layers でオブジェクトを整理

6. Bone でスケルトン作成 → Assign to Mesh

7. Weight で各ボーンの影響範囲を塗る

8. Anim でポーズ → キーフレーム記録 → プレビュー

9. GLB Export でファイル出力
